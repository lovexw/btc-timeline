# 比特币大事记网站

一个专业、客观、易读的比特币历史时间线网站，记录从比特币诞生到现在的完整发展历程。

## 🎯 项目目标

- **时间线可视化**: 清晰展示比特币发展的关键节点
- **结构清晰**: 按时间顺序和主题分类组织内容
- **面向大众易懂**: 使用简洁明了的语言解释复杂概念
- **保持中立记录**: 客观记录历史事实，不夸大、不煽动

## 📁 项目结构

```
bitcoin-timeline/
├── index.html          # 主页面文件
├── styles.css          # 样式文件
├── data.js            # 比特币历史数据
├── script.js          # 页面交互逻辑
└── README.md          # 项目说明文档
```

## 🚀 功能特性

### 📊 主要内容模块

1. **概览**: 比特币发展的关键数据统计
2. **时间线**: 完整的历史事件时间线（2008-2025）
3. **重要里程碑**: 按年份分类的关键事件
4. **技术发展**: 技术升级和创新历程（包括BIP32/BIP39等重要提案）
5. **价格历史**: 价格变化的重要节点
6. **重要新闻**: 影响行业发展的重大新闻
7. **挖矿历程**: 从CPU到ASIC的完整挖矿发展史
   - 经典矿机发布（Avalon、蚂蚁S1/S3/S9/S19系列等）
   - 挖矿难度里程碑（1到90T+的完整记录）
   - 算力迁移事件（中国禁矿等）
8. **政策监管**: 中美等国家政策演变
   - 中国监管历程（2013首次监管→2021全面禁止）
   - 美国政策发展（FinCEN→IRS→ETF获批→战略储备）
   - 其他国家政策（萨尔瓦多法币、欧盟MiCA等）
9. **安全事故**: 重大漏洞与黑客事件
   - 协议漏洞（184亿BTC漏洞、CVE-2018-17144等）
   - 交易所被盗（Mt.Gox、Bitfinex、币安等）

## 📅 时间线覆盖范围

### 创世时期 (2008-2010)
- 白皮书发布
- 创世区块诞生
- 首笔交易完成
- 披萨日事件
- 184亿BTC漏洞修复
- GPU挖矿开启

### 早期发展 (2011-2013)
- 交易所兴起
- 价格突破关键点位
- 第一次减半
- 主流关注
- ASIC矿机诞生（Avalon、蚂蚁S1）
- BIP32/BIP39标准提出
- 中国首次监管

### 成长期 (2014-2016)
- Mt. Gox事件
- 第二次减半
- 技术升级（BIP65、SegWit提案）
- 监管关注
- 经典矿机S9发布

### 繁荣期 (2017-2020)
- 价格历史新高（近2万美元）
- 分叉事件（BCH）
- SegWit激活
- 中国禁止ICO和交易所
- 闪电网络上线
- 机构入场（MicroStrategy、PayPal）
- 第三次减半
- 百T矿机S19发布

### 主流化 (2021-2025)
- 历史最高价格（6.9万美元）
- 合规化进程（Coinbase上市）
- 中国全面禁止挖矿（算力迁移、难度暴跌28%）
- 萨尔瓦多法币采用
- Taproot升级
- 现货ETF获批（2024）
- 第四次减半（2024）
- 挖矿难度突破90T
- 美国战略储备计划（2025）
