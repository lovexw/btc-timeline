# 比特币大事记网站

一个专业、客观、易读的比特币历史时间线网站，记录从比特币诞生到现在的完整发展历程。

## 🎯 项目目标

- **时间线可视化**: 清晰展示比特币发展的关键节点
- **结构清晰**: 按时间顺序和主题分类组织内容
- **面向大众易懂**: 使用简洁明了的语言解释复杂概念
- **保持中立记录**: 客观记录历史事实，不夸大、不煽动

## 📁 项目结构

```
bitcoin-timeline/
├── index.html          # 主页面文件
├── styles.css          # 样式文件
├── data.js            # 比特币历史数据
├── script.js          # 页面交互逻辑
└── README.md          # 项目说明文档
```

## 🚀 功能特性

### 📊 主要内容模块

1. **概览**: 比特币发展的关键数据统计
2. **时间线**: 完整的历史事件时间线（2008-2024）
3. **重要里程碑**: 按年份分类的关键事件
4. **技术发展**: 技术升级和创新历程
5. **价格历史**: 价格变化的重要节点
6. **重要新闻**: 影响行业发展的重大新闻

### 🎨 设计特点

- **响应式设计**: 适配桌面和移动设备
- **现代化界面**: 简洁美观的视觉设计
- **流畅动画**: 页面滚动和元素加载动画
- **交互体验**: 搜索、导航、主题切换等功能

### 🔧 交互功能

- **智能搜索**: 快速查找特定事件
- **平滑导航**: 点击导航平滑滚动到对应区域
- **主题切换**: 支持明暗主题切换
- **返回顶部**: 快速返回页面顶部
- **键盘快捷键**: 支持常用快捷键操作

## 📅 时间线覆盖范围

### 创世时期 (2008-2010)
- 白皮书发布
- 创世区块诞生
- 首笔交易完成
- 披萨日事件

### 早期发展 (2011-2013)
- 交易所兴起
- 价格突破关键点位
- 第一次减半
- 主流关注

### 成长期 (2014-2016)
- Mt. Gox事件
- 第二次减半
- 技术升级
- 监管关注

### 繁荣期 (2017-2020)
- 价格历史新高
- 分叉事件
- 机构入场
- 第三次减半

### 主流化 (2021-2024)
- 历史最高价格
- 合规化进程
- 现货ETF获批
- 第四次减半
